Previsão de Chuva na Austrália

Aplicação full stack para prever se vai chover no dia seguinte com base em dados meteorológicos do dataset weatherAUS.

Predição por linha CSV

Cole uma linha completa do dataset original, separada por vírgulas. O sistema irá interpretar os campos automaticamente na ordem correta.

Predição por formulário

Preencha os campos manualmente para testar o modelo diretamente com os valores meteorológicos.
